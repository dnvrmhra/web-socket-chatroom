# Real-Time Chat Application

A full-stack real-time chat application built using React, Node.js, Express, and Socket.io.
This project demonstrates real-time communication between multiple users with a clean and responsive interface.

## Features

* Username-based login (Join Chat)
* Real-time messaging using WebSockets
* Live online users list
* Typing indicator ("User is typing...")
* System messages (user join/leave)
* Message timestamps
* Auto-scrolling chat window
* Clean and responsive UI


### Frontend

* React.js
* CSS (Custom styling)

### Backend

* Node.js
* Express.js
* Socket.io
